What does each parameter do?
w — war prefactor. War rate between neighbours is w·B_ij·min(S_i, S_j): a dyad fights at a rate set by the weaker side's strength. Higher w → more wars; mutual growth also drives more wars.
c — fragmentation prefactor. Frag rate is c·A·(1−S/A): large under-strength territories collapse internally (Khaldun's asabiyya). Drag fully left to set c = 0 and disable fragmentation entirely.
A* — Khaldun ceiling: a state's strength saturates at A* no matter how large its territory. Smaller A* → big states are more brittle.
L, N₀ — lattice size and initial number of states.
Reaction channels (v2)
R1 Growth: S_i → S_i+1 at rate max(0, C_i − S_i) with C_i = A_i / (1 + A_i/A*).
R2 War i↔j: rate w·B_ij·min(S_i,S_j). Winner drawn with probability S_i/(S_i+S_j); one boundary site of the loser is taken, weighted by the number of edges that site shares with the winner (bond-percolation rule — keeps territory compact). Both pay ½·min(S_i,S_j) strength.
R3 Fragmentation: rate c·A_i·(1−S_i/A_i)_+. Two random seeds picked inside the territory; each cell goes to the nearer seed (periodic L2). Both new fragments start with S = 0. Setting c = 0 disables this channel entirely.
